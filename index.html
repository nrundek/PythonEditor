Quick instructions

Show or hide usage instructions
  1. In the File name field, enter a name without an extension or with the .py extension.
  2. To load an existing program, press Open existing .py file.
  3. The current line and column are announced after pressing Enter, after using the Announce position button, or with the Alt+L shortcut while the focus is in the Python code field.
  4. For quick navigation, you can enter a number in the Go to line field and press Enter or the Go button.
  5. Press Run code. The program will run in the browser.
  6. If the program uses input(), the browser will open a dialog box for entering values.
  7. The result or error description will be displayed in the Terminal output area.
  8. The code is checked line by line, so errors will be displayed one at a time. When an error is fixed and the code is run again, the next error (if any) will be displayed.
  9. The Save file button saves the current editor content as a .py file.
  10. The file will be saved to the default Downloads folder. From there, move it to this editor’s folder if you want to run it with the .bat file in the system terminal with Python installed.
